# Non-English Libraries

Libraries with names in non-English languages or joke/esoteric naming.

## Subdivisions

- **NonEnglish-1**: Asian languages (Japanese, Chinese, Korean, etc.)
- **NonEnglish-2**: European languages (French, German, Spanish, Russian, etc.)
- **NonEnglish-3**: Others (Arabic, Hebrew, constructed languages, etc.)

## Note

These are joke or esoteric libraries with non-English names for variety and fun.
