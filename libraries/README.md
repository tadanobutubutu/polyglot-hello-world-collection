# ⚠️ Libraries Directory - Migration in Progress

## Status

This directory contains ~8,000 library files but **GitHub truncates the view to 1,000 files**.

You may see the message:
> "Sorry, we had to truncate this directory to 1,000 files. 2,264 entries were omitted from the list."

## New Organization

Libraries are being reorganized alphabetically:

- `libraries/a/A-1/` through `libraries/a/A-4/` - A libraries
- `libraries/b/B-1/` through `libraries/b/B-4/` - B libraries
- ...
- `libraries/s/S-1/` through `libraries/s/S-4/` - S libraries (largest)
- ...
- `libraries/z/Z-1/` through `libraries/z/Z-2/` - Z libraries
- `libraries/symbols/` - Symbol/numeric libraries
- `libraries/nonenglish/` - Non-English named libraries

## How to Browse

1. Go to [README.md](../README.md)
2. Open the Libraries section
3. Click on a subdivision link (e.g., [S-3] for libraries Se-So)
